# **App Name**: NEU Library

## Core Features:

- Welcome Message Display: A prominent and centrally located welcoming message for the library application.
- Dynamic Background Image: Ability to display a customizable background image behind the main content.
- Login Action Button: A button to navigate users to the login screen for authenticated access.
- Sign Up Action Button: A button to guide new users to the account registration page.
- Responsive Layout: The page layout adapts seamlessly to various screen sizes, from mobile devices to desktop monitors.

## Style Guidelines:

- Primary color: A deep, intelligent blue (#5164E5) to evoke knowledge and serenity, suitable for key UI elements and active states.
- Background color: A very dark, subtle blue-grey (#1E2024) providing a focused and minimalistic backdrop for content.
- Accent color: A vibrant, clear blue (#77CDF0) for calls-to-action and important highlights, ensuring high contrast against the dark background.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable appearance across all devices, maintaining a minimalistic feel.
- Minimal use of clean, line-based icons for UI elements, supporting the minimalistic design approach.
- Centralized content focus, with ample negative space around elements to enhance readability and a clean aesthetic. Buttons strategically placed for easy access.
- Subtle and functional micro-animations, such as hover effects on buttons, to provide user feedback without being distracting or heavy.
